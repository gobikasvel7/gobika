The Study Abroad Companion MVP is a frontend-first web application designed to demonstrate a unified, student-centric study-abroad support platform. This prototype focuses on usability, clarity, and core student workflows
Centralized Student Dashboard Displays key study-abroad metrics such as selected country, application count, offers received, and visa status
University Discovery Module Country-based university search with instant results to support early-stage decision-making
Application Progress Tracker Visual step-by-step tracking of the study-abroad journey from profile creation to visa approval.
Rule-Based AI Assistant (Demo)Logic-driven chat assistant that provides instant study-abroad guidance, showcasing readiness for AI integration.
Google Maps Integration Interactive map view to explore university locations geographically
Multilingual Accessibility Google Translate integration to support multiple languages and improve inclusivity.
Modern, Student-Centric UI Clean card-based design with intuitive navigation for an improved user experience.
Built as a rule-based MVP to validate core ideas quickly
Designed with a clear upgrade path to AI-driven personalization
Frontend-first approach enables rapid iteration and user feedback
Architecture is cloud-ready and scalable for future backend integration
Future Enhancements (Planned):AI-powered recommendations using Vertex AI
Backend integration for real-time data and persistence
Secure document upload and verification
Peer and alumni mentorship features 
Advanced financial planning tools
